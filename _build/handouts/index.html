Data Wrangling Project

Introduction

In this project, you will choose a dataset and practice your data manipulation skills using Python and pandas.

Choose Dataset

Spend some time exploring datasets at DataSF: https://datasf.org/opendata/

Then, choose a dataset you would like to use for this exercise.

Dataset Requirements

  • Be sure that the dataset that you choose provides data in .csv format.

  • Your dataset must include both numeric and textual data.

When you have chosen, download the dataset.

Data Wrangling Practice

Using your lecture videos to guide your syntax, conduct the following steps of data manipulation using pandas. It will be helpful for you to follow along with the steps presented in the video Pandas Playbook: Manipulating Data for this exercise.

Write code in a Python file

Be sure to write your code in a Python file that you can push to Github at the end. Do not use Jupyter notebook during this exercise. We will be covering Jupyter notebook later on in the course, but until then, continue to practice writing code in a regular .py file.

Step 1: Exploration

Write a function in Python that performs exploratory data analysis on your dataset. Use pandas to print basic information about your dataset such as number of rows, averages for numeric columns, and any other useful explanatory information about your dataset.

Step 2: Select, Filter, Sort

Using dataframes in pandas, write 2-3 functions that filter and sort your dataset. For example, write a function that produces a subset of your data that includes a numeric value that is higher than the average numeric value for that column. Another example: write a function that sorts the rows alphabetically based on a texual column.

Step 3: Clean Data

Identify whether your dataset has any missing or null value using pandas. If there are null values in a numeric column, determine whether you should remove the data, or if there is a reasonable replacement for null, such as replacing it with a 0 value.

Additionally, identify if there are any low/high outliers in your dataset using pandas.

Step 4: Transform Data

Conduct 1-2 data tranformations on your dataset, such as indexing your data with a useful numeric value, or combining multiple columns into a new column.
